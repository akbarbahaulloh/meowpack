### 3. Panduan Lengkap Auto Share
**File:** [PANDUAN-AUTOSHARE.md](PANDUAN-AUTOSHARE.md)

**Isi:**
Tutorial step-by-step untuk 10 platform:

1. ✈️ **Telegram** - Bot API
   - Cara buat bot
   - Dapatkan Chat ID
   - Konfigurasi

2. 📘 **Facebook** - Graph API
   - Buat Facebook App
   - Generate Page Access Token
   - Setup permissions

3. 📸 **Instagram** - Business API
   - Hubungkan ke Facebook Page
   - Dapatkan Instagram Business ID
   - Requirements gambar

4. 🐦 **X (Twitter)** - OAuth 1.0a
   - Buat Developer Account
   - Generate API Keys (4 credentials)
   - Set permissions

5. 💼 **LinkedIn** - UGC Posts API
   - Buat LinkedIn App
   - Generate Access Token
   - Dapatkan Author URN

6. 🦋 **Bluesky** - AT Protocol
   - Buat App Password
   - Setup (paling mudah!)

7. 🧵 **Threads** - Meta Threads API
   - Dapatkan Threads User ID
   - Setup token

8. 📌 **Pinterest** - Pins API
   - Buat Pinterest App
   - Generate token
   - Dapatkan Board ID

9. 💬 **Line Notify** - Notify API
   - Generate token (super mudah!)

10. 📱 **WhatsApp** - Business Cloud API
    - Setup WhatsApp Business
    - Verifikasi nomor
    - Konfigurasi

**Untuk:** Pengguna yang ingin setup platform tertentu

**Waktu baca:** 30-60 menit (tergantung platform)

## 🎯 Panduan Berdasarkan Kebutuhan

### Saya Baru Install Plugin
1. Baca [README.md](README.md) - Overview
2. Aktifkan fitur di **MeowPack → Pengaturan**
3. Pilih 1-2 platform mudah dari [Quick Start](QUICK-START-AUTOSHARE.md)

### Saya Ingin Setup Telegram
1. Buka [Panduan Auto Share](PANDUAN-AUTOSHARE.md#-1-telegram)
2. Ikuti step-by-step (5 menit)
3. Test koneksi

### Saya Ingin Setup Facebook/Instagram
1. Baca [Panduan Facebook](PANDUAN-AUTOSHARE.md#-2-facebook)
2. Baca [Panduan Instagram](PANDUAN-AUTOSHARE.md#-3-instagram)
3. Perlu 15-20 menit setup
4. Token perlu di-extend setiap 60 hari

### Auto Share Saya Error
1. Cek error code di meta box post
2. Buka [Troubleshooting](TROUBLESHOOTING-AUTOSHARE.md)
3. Cari error code Anda (401, 403, 400, 429)
4. Ikuti solusi

### Saya Ingin Setup Semua Platform
1. Mulai dari yang mudah:
   - ✅ Telegram (5 menit)
   - ✅ Bluesky (3 menit)
   - ✅ Line Notify (3 menit)
2. Lanjut yang menengah:
   - ⚙️ Twitter (10 menit)
   - ⚙️ Facebook (15 menit)
3. Terakhir yang advanced:
   - 🎓 Instagram (20 menit)
   - 🎓 LinkedIn (20 menit)
   - 🎓 Pinterest (20 menit)
   - 🎓 Threads (20 menit)
   - 🎓 WhatsApp (30 menit)

**Total waktu:** 2-3 jam untuk semua platform
